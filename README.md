## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jeiku/javascript-pacman.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```

3. Start game on local host
   ```sh
   npm start
   ```
